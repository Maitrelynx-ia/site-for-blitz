Pour que le site fonctionne correctement

Ce site utilise l'API officielle api.wotblitz.eu/wotb/ avec l'ID: 0cd52ad2ab52ea7511013106881cc3f7.

Pour éviter les erreurs CORS, vous devez :

  1. Ouvrir ce fichier via un serveur web local (méthode recommandée)
  2. Ou héberger ce fichier sur un site web (GitHub Pages, Netlify, etc.)
  3. Ou utiliser une extension navigateur pour désactiver CORS
python -m http.server 8000

Puis ouvrez http://localhost:8000 dans votre navigateur
